####  Definition of web server
1. In light of *hardware*, web server is a **computer** which can **store** web server software and website file, it can **connect** to internet and it can be physically replaced by other website equipment.
2. In light of *software*, web server contains many files which **manage how the web user surfs** ( It contains **HTTP server**  at least. )
> HTTP server is a part of web server, it understands *URLs* and *HTTP*, it can surf hosted website and sent the content to end-user's equipment, like browser.
